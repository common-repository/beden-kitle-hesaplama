== Description ==

Bu eklenti ister widgetler yardımı ile rahatlıkla sidebara yerleştirilebilir. İsterseniz kısakodu sayesinde istediğiniz bir sayfaya PHP kodu veya kısakodu ile gömülebilir. Eklenti sayesinde kullanıcılarınız kendi beden kitle endekslerini hesaplayabilecekler.

== Installation ==

1. `beden-kitle-hesaplama` klasörünü `/wp-content/plugins/` dizinine yükleyiniz
2. Beden Kitle Endeksi Hesaplama Aracı eklentisiniz wordpress admin menüsündeki 'Eklentiler' menüsünde aktive ediniz.
3. Eklentide herhangi bir ayar paneli bulunmamaktadır.
4. [bkeha] - Bu kodu wordpress sayfa yada postunun içine istediğiniz yere kopyala/yapıştır yaparak canlı skorları sayfanızda gösterebilirsiniz.
